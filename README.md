# Cách sử dụng

### Thêm file .env

```
MONGO_URL= 

PORT = 5000
```

### Install back-end dependencies

```
npm install
```
### Tạo thư mục client
- Clone dao diện client về.

### Install client dependencies

```
cd client
npm install
```

### Run both Express & React from root

```
npm run dev
```
